The Problem

AEC's most expensive
mistake happens at the desk.

Construction rework costs India over ₹50,000 Cr annually — and most of it starts with drawing errors and compliance failures no one caught in time.

01
Drawing review is manual and slow

A 200-sheet drawing set takes 2–3 days to review by hand. Missed dimension clashes, incomplete annotations, and referencing errors slip through — arriving on site at 10× the cost to fix.

02
Compliance expertise is locked in silos

Building codes change constantly across jurisdictions. NBC, IS codes, state bylaws, RERA requirements — firms rely on a single expert's knowledge. One error means a rejected submission and weeks of delay.

03
BIM data stays underutilised

Firms invest in Revit and generate data-rich BIM models — then query nothing. Clash detection is the ceiling. Material schedules, code checks, and design intelligence sit locked in the model file.

Product

Four modules.
One intelligent platform.

eclipix sits between your design tool and your downstream workflow — reading every drawing, checking every clause, querying every model.

Drawing Analysis
DrawScan

Upload any PDF or DWG drawing set and eclipix extracts annotations, validates dimensions, flags missing callouts, and identifies incomplete sheets — in minutes, not days.

  • Dimensional inconsistency detection across plan, elevation, section
  • Missing annotation flags — fire ratings, structural grids, accessibility paths
  • Incomplete drawing set detection with severity-tiered reports
  • Drawing Health Score with Critical / Major / Minor classification
Compliance Intelligence
CompliCheck

A continuously updated regulatory knowledge base covering NBC 2016, state amendments, IS codes, accessibility standards, and fire NOC requirements — checking your drawings clause by clause.

  • NBC 2016 + 12 state jurisdictions — room sizes, corridors, egress
  • IS 456, IS 800 structural drawing compliance
  • RPWD Act accessibility checks — ramps, door widths, parking
  • Jurisdiction-specific report formatted for submission
BIM Intelligence
BIMind

Connect your Revit or ArchiCAD model and query it in plain language. eclipix adds a natural language analytics layer on top of your BIM data — no exports, no scripting.

  • NL queries: "All fire doors with rating < 60 min in east wing"
  • Automated BOQ extraction with specification attributes
  • Clash intelligence — severity-classified with resolution suggestions
  • Design change tracking between model versions
Project Intelligence
AuditTrail

eclipix becomes the audit backbone of every project — tracking submissions, re-runs, markups, and approvals. Automated RFI drafting turns every flagged issue into an actionable document.

  • Version-linked compliance — re-runs on every drawing revision
  • Automated RFI drafting from flagged issues
  • Project health score + compliance readiness dashboard
  • API integration — Procore, Aconex, BIM 360
